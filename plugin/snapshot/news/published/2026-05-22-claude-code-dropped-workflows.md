Anthropic introduced a new /workflows feature in Claude Code 2.1.147 that replaces the LLM orchestrator with code-based workflows, improving multi-agent system efficiency by avoiding token window bloat. This allows structured phases, parallel execution, conditionals, retries, and background runs, but the feature was temporarily taken down shortly after release.
