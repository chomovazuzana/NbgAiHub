A repository (repo) is where a single project lives on GitHub. One folder, one project. It holds the files, the full history of every change ever made, the issue tracker, and the PR review surface.

The team's repos live under `github.com/orgs/556LowCodeNoCode/repositories`. New joiners get read access on day one; specific contributor access is added per repo when needed.

You'll spend most of your time inside *one* repo at a time, with Claude Code running in that folder.
