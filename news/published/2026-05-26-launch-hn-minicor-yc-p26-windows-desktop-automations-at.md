Minicor is a platform that enables scalable Windows desktop automation using Python-based RPA workflows controlled by an MCP that integrates with Claude Code. It addresses common challenges in desktop RPA such as scripting complexity, orchestration, debugging, and failure rates by providing version-controlled code workflows, VM cloning, 2FA handling, and integration with Slack and LLM verification.
